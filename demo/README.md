# Demo video

Short local walkthrough of the Oracle Property Intelligence PR.

## File

- `oracle-property-intelligence-demo.webm` (~35s, 1366×768)

## Scenes

1. Dashboard — pipeline stats
2. Run summary — six source types
3. About — IPFS artifacts
4. Sandbox — roof age filter (interactive)
5. Sandbox — transit distance + Palo Alto city
6. Explore — README demo questions
7. Agent — MCP prompts

## Re-record

```bash
./scripts/run.sh ui-start          # terminal 1
./scripts/run.sh record-demo       # terminal 2
```

Output is written here as `oracle-property-intelligence-demo.webm`.

Open with any browser or VLC:

```bash
xdg-open demo/oracle-property-intelligence-demo.webm
```
